# 🌾 Smart Agriculture System
A web-based Smart Agriculture System that helps farmers make better decisions using technology. This application provides crop recommendations, fertilizer suggestions, and future scope for weather-based alerts and yield prediction.

# Features
▪️🌱 Crop Recommendation
   ▪️Suggests suitable crops based on soil type and season
▪️🧪 Fertilizer Suggestion (Planned / Extendable)
   ▪️Recommends fertilizers based on crop and soil nutrients
▪️🌦️ Weather Alerts (Future Scope)
   ▪️Alerts for rain, temperature, and irrigation planning
▪️📊 Yield Prediction (Future Scope)
   ▪️Estimates crop production based on inputs


# 🛠️ Tech Stack
▪️Backend: Java, Spring Boot
▪️Frontend: Thymeleaf, HTML, CSS, Bootstrap
▪️Database: MySQL
▪️Tools: Maven, IntelliJ

# Future Enhancements
